# ORCID icon

**\[deprecated\]**

Please use
[`icn_orcid()`](https://arnaudgallou.github.io/plume/dev/reference/icn_orcid.md)
instead.

## Usage

``` r
orcid(size = 16, bw = FALSE)
```

## Arguments

- size:

  Size of the icon (in pixels).

- bw:

  Should the black and white version of the icon be used?

## Value

A plume icon.
